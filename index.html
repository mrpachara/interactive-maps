In properties attribute:

  • name: string — The name of point.
  • actionUrl: string — The URL of playing this point.
  • icon: string — The URL for point icon.
  • iconSize: [number, number] — The size of icon [width, height]. Default is [64, 64].
  • iconAnchor: [number, number] — The icon anchor position [x, y] from icon bottom-left. Default is [width / 2, height / 2].
  • popupAnchor: [number, number] — The popup anchor position [-x, -y] from icon anchor position. Default is [0, -iconAnchor[1]].
  • status: 'unvisited' | 'remote-visited' | 'visited' — The status of point.
  • stampIcon: string — The URL of stamp icon. Default depends on status.
